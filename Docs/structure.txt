!Внимание! Структура программы будет дополнена!

1. Ядро
    Ядро программы должно обеспечитвать полную функциональность работы продукта (программы) при подключении любого
    интерфейса взаимодействия с пользователем (консоль, графический интерфейс, чат-бот).

2. Модуль работы с базой данных
    Модуль, предназначенный для выполнения запросов в базу данных

3. Телеграм-бот
    Модуль взаимодействия с пользователем через телеграм-бота

4. Консольная программа
    Модуль взаимодействия с пользователем через консоль (терминал)
